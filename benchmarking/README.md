# Benchmarking datasets

⚠️ **Synthetic demo data** from the specification — used to demonstrate the
benchmarking pipeline in `notebooks/05_QPU_Benchmarking.ipynb`. These are NOT real
measurements; the real verified results live in `data/quantum_results_verified.csv`
(Zenodo DOI 10.5281/zenodo.20749395).

- `qaoa_maxcut_results.csv` — synthetic QAOA MaxCut scores (classical/quantum/AI)
- `vqe_h2_results.csv` — synthetic H2 energies per bond length
