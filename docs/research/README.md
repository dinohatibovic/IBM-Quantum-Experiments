# Research documents — status: DRAFTS / TEMPLATES

⚠️ **Important before submitting anywhere (arXiv, Zenodo, IBM):**

The documents in this folder are **templates and drafts**. The numeric "results"
mentioned in them (e.g., "convergence improves by 18%", "VQE error reduced by
0.004 Ha", "12% better MaxCut score") are **placeholder values from the template —
those experiments have not been performed** and these numbers MUST NOT go into a
real publication or application until replaced with real, reproducible measurements.

The same applies to placeholder contacts (`contact@qoptisolve.ai`, missing ORCID) —
replace them with real data.

Real, verified results exist only in `data/quantum_results_verified.csv`
(this repository, Zenodo DOI 10.5281/zenodo.20749395) and should be the
only source cited.
