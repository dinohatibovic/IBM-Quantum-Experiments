\section{IBM Quantum Hardware}

Experiments were executed on:
\begin{itemize}
    \item \textbf{ibm\_fez Heron R2} — 156 qubits, low-noise, high-fidelity backend,
    \item \textbf{ibm\_torino Heron R1} — 133 qubits, baseline Heron-class device.
\end{itemize}

Both backends support:
\begin{itemize}
    \item Qiskit Runtime SamplerV2,
    \item OpenQASM 2.0,
    \item QPY serialization,
    \item BitArray binary archives.
\end{itemize}
